1.
사용자와 컴퓨터가 가위바위보하는 프로그램을 작성하려고 한다. 
가위,바위,보는 1,2,3의 숫자로 나타냅니다.
컴퓨터는 랜덤으로 손을 선택합니다. 결과화면에는 ‘무승부’, ‘사용자승리’ ,‘사용자패배’ 중 하나를 표시합니다. 무승부의 경우는 다시 승부합니다.
 (승패가 나면 사용자의  승리 또는 패배를 출력한다 )

🧩
📝사용자의 prompt 입력을 받는다(가위1, 바위2, 보3 중 택 1 입력)
📝컴퓨터도 마찬가지로 랜덤으로 하나를 출력하도록 입력
📝무승부, 컴퓨터 승부, 사용자 승부가 가능한 모든 경우의 수를 if else문으로 작성
📝결과에 따른 멘트가 출력되도록 입력


 

2. 3리터짜리 양동이와 5리터짜리 양동이를 하나씩 들고 강에 가서, 딱 4리터의 물을 길어 오려면 어떻게 하면 좋을까요?
  - 다른도구는 없습니다.

  🧩
  📝각(3리터, 5리터)양동이가 빈 채로 시작
  📝5리터 양동이의 물을 3리터 양동이에 가득 채움(5리터 양동이엔 2리터가 남음)
  📝3리터 양동이 물을 버려준다 
  📝5리터 양동이에 남아있던 2리터의 물을 비어있는 3리터짜리 양동이에 다시 붓는다
  📝비어있는 5리터 양동이에 물을 다시 가득 채워서 2리터만 남아있는 3리터짜리 양동이에
     물을 가득 채운다(이 과정에서 정확하게 1리터만 빼낼 수 있음)
  📝정확히 1리터만 빠져나간 5리터 양동이만 남게된다 = 딱 4리터의 물




3. 수 맞추기 게임
   컴퓨터가 임의의 수를 내면 사람이 그 수를 맞추는 게임이다 . 맞출 때 까지 반복합니다
   (컴퓨터는 100이내의 임의의 수를 낸다)
   up & down으로  컴퓨터가 낸 수를 맞추기 

 
   추가옵션사항 : 3번 안에 맞추는 맛있는 간식이 제공된다. ( 간식을 받는 사람에게는 적절한 메시지를 출력한다)

   🧩
   📝prompt를 이용해서 사용자의 답을 받는 창을 띄운다."1~100 중 하나의 숫자를 적어주세요"
   (1~100이내, 하나의 수만!)
   📝사용자가 답을 입력하는 횟수를 카운트한다
   📝컴퓨터가 1~100 이내의 수를 랜덤으로 뽑도록 let으로 변수를 선언해준다 
   📝사용자가 입력한 수가 컴퓨터가 뽑은 랜덤의 수보다 큰 경우 => down 출력
   📝사용자가 입력한 수가 컴퓨터가 뽑은 랜덤의 수보다 작은 경우 => up 출력
   📝사용자가 입력한 수가 컴퓨터가 뽑은 랜덤의 수와 같을 경우 =>
      1. 3번 안에 정답을 맞췄을 경우 => "정답! 3번 안에 맞췄습니다! 맛있는 간식이 제공됩니다!" 출력
      2. 3번을 넘겨서 정답을 맞췄을 경우 => "정답! 축하드립니다!" 출력

 🧩

    =>let 
 📝컴퓨터가 랜덤한 숫자 중 하나를 뽑도록 let으로 선언한다.



5. 자신의 나이구하기

🧩
📝prompt를 이용해서 사용자가 태어난 연도를 입력하는 창을 띄운다.

📝let을 이용하여 오늘 날짜에서 사용자가 입력한 연도를 빼주는 변수를 선언한다.
   => let age = todayYYYY - prompt(YYYY);
📝계산해서 나온 결과가 창에 뜨게 입력해준다 
   => alert("당신의 현재 나이는 + age + 입니다");
